属性

グローバル属性

グローバル属性のみ使用することができます。

value

ol要素の子要素である場合にのみ指定できる属性です。左側に表示されるリストの値を指定することができます。

使用例

コーヒーの淹れ方を例にしたHTMLサンプルコード

li要素の中にol要素やul要素を入れて階層構造にすることが可能です。
また、ol要素の子要素に限りvalue属性を使用することができます。

<ul>
	<li>ペーパードリップの淹れ方
		<ol>
			<li>手順1</li>
			<li>天順2</li>
			<li value="3">補足</li>
			<li value="3">補足</li>
		</ol>
	</li>
	<li>フレンチプレスの淹れ方
		<ol>
			<li>手順1</li>
			<li>天順2</li>
			<li>天順3</li>
		</ol>
	</li>
</ul>
