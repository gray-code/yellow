グローバル属性

グローバル属性のみ使用することができます。

使用例

PHPコードを表したい場合は、下記のようになります。変数についてはvar要素を使うことができます。
コード内にHTML要素を書くとHTML出力されてしまうため、「<」は「&lt;」、「>」は「&gt;」と書く必要があることに注意してください。

<code class="language-php">&lt;?php
var $animals = array(
	"dog" =&gt; "pochi",
	"cat" =&gt; "tama",
	"fox" =&gt; "konn"
);

foreach( $animals as $value ) {
	var_dump($value);
}
?&gt;</code>
